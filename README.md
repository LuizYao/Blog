# Blog

本博客采用 [Hugo](https://gohugo.io/) 构建，主题为 [Clarity](https://github.com/chipzoller/hugo-clarity)。

# 使用

克隆仓库：

```bash
$ git clone --recursive https://github.com/luizyao/blog.git
```

本地构建：

```bash
$ hugo -D server
```
